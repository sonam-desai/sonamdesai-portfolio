  • Client: Academic SDSU
  • Project date: Fall 2023
  • Project URL: Code Link for Data Exploration

The ongoing project focuses primarily on cloud and information security and entails the complete setup and operation of an online coffee shop website on the Amazon Web Services (AWS) platform. Initially, high availability and durability are provided by using Amazon S3, a scalable object storage service, to host static website files. Meanwhile, to demonstrate the flexibility of AWS services, a managed and secure backend for the online coffee shop's database is being explored for Amazon RDS, a relational database service. An Amazon EC2 instance is deployed as the web server, which provides flexibility and control over compute capacity and is a key component of the project. Key responsibilities include setting up HTTPS for secure communication, configuring access controls via IAM, and guaranteeing data protection for the website and database via encryption techniques.
