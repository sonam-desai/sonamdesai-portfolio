The goal for this study is to understand the vaccination progress around the world and identify which countries have the maximum people vaccinated. The methods used to analyse the data are using the python and creating visualizations using the visualization tool - Tableau. For Python, we have created a dataframe and used packages like Numpy, Pandas, Seaborn and Matplot to analyse and create visualizations.
